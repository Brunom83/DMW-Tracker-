📋 DESCRIÇÃO DO PROJETO
DMW Farm Tracker é uma aplicação web desenvolvida especificamente para jogadores do Digimon Masters World (DMW) - um servidor privado de Digimon Masters Online. A aplicação ajuda os jogadores a organizar e optimizar o seu tempo de jogo através de um sistema completo de tracking de farm, moedas e progresso.

🎯 OBJETIVO PRINCIPAL
Ajudar os jogadores de DMW a:

Controlar ganhos de moedas (Tera, Mega, Bits) em sessões de farm

Organizar horários de tours automáticos do jogo

Calcular eficiência de diferentes métodos de farm

Planear objetivos de crafting e progressão

Maximizar o tempo de jogo com métricas e estatísticas

🚀 FUNCIONALIDADES PRINCIPAIS
💰 SISTEMA DE MOEDAS
Controlo "Antes/Depois" de sessões de farm

Cálculo automático de ganhos em Tera/Mega/Bits

Conversor inteligente entre as três moedas do jogo

Registo histórico de todas as sessões

🥚 CALCULADORA DE CRACKED EGGS
7 tipos diferentes de Cracked DigiEggs com valores reais

Cálculo em tempo real do total em Bits/Mega/Tera

Gestão de inventário de eggs coletados

Valores actualizáveis conforme a economia do jogo

🏹 SISTEMA DE TOURS
Forest Tour - 8 raids × 300M = 2.4T + seals aleatórios

Bless Tour - 3 mapas × 0.9T = 2.8T + seals

Alertas inteligentes com notificações sonoras

Horários personalizáveis para cada tour

Histórico completo de tours realizados

📊 DASHBOARD INTELIGENTE
Métricas em tempo real do progresso diário

Estatísticas de eficiência

Próximos tours agendados

Actividade recente timeline

🛠 TECNOLOGIAS UTILIZADAS
Frontend: HTML5, CSS3, JavaScript (ES6+)

Framework: Bootstrap 5.3.0

Ícones: Font Awesome 6.4.0

Armazenamento: LocalStorage (dados locais)

Design: Glass morphism, Dark theme, Design responsivo

🎮 CONTEXTO DO JOGO (DMW)
O DMW é um MMORPG baseado no universo Digimon que requer:

Farm intensivo para obter moedas e itens

Gestão de tempo para participar em eventos automáticos (tours)

Planificação para crafting de equipamentos end-game

Controlo de recursos devido à economia complexa (Tera←→Mega←→Bits)
